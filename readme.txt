== Description ==

Simple and elegant cookie consent is all you need. <strong>Cookie Accept</strong> is a simple cookie consent, concern and compliance. Get your site ready for GDPR, CCPA or the others. Control your cookie consent via options with live preview.

## Easy to Customize!

Here some key features you will get:

* <strong>Live Preview.</strong> See your cookie consent while editing it.
* <strong>Dark Mode Ready.</strong> Match your site color mode.
* <strong>Custom Cookie Consent Message</strong> Yes, you can customize the message.
* <strong>Clean CSS and Javascript Frontend.</strong> If your visitor look at your site source code, they possibly don't realize you use this plugin.
* <strong>Single Line Database.</strong> Yep! this plugin save all your custom settings of this plugin into only one row of data inside <code>wp_options</code> table.
* <strong>Responsive.</strong> Well it is a must for website nowadays.
